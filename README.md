# Nua
A small & simple single file library for implementing neural networks in Lua

> [!WARNING]
> Unstable software. Work in progress.

## Disclaimer
This is an amateur project.
This project was created for me to learn about neural networks and Lua.
Suggestions on code improvements are welcome.

## Requirements
- \>=lua5.4

> lua5.3 and below is untested. Contributions are welcome.

## Getting Started
Only nua.lua is required to use this library

1. Clone this repository
    ```
    git clone https://github.com/phyllis-ch/nua.git
    ```
    or
    ```
    wget https://raw.githubusercontent.com/phyllis-ch/nua/refs/heads/master/nua.lua
    ```

2. Include the library
    ```lua
    local nua = require("nua")
    
    ...
    ```

## Examples
Examples are located in the examples directory.
Currently a XOR gate is provided as an example.
